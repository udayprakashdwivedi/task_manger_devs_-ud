## **Requirements**
- **Ruby Version**: `3.2.1`
- **Rails Version**: `7.x`
- **Database**: PostgreSQL 
- **Dependencies**:
  - `Devise` & `Devise-JWT` (Authentication)
  - `Pagy` (Pagination)
  - `ActiveModel::Serializers` (JSON serialization)

## Setup the database 
 * rails db:create
 * rails db:migrate
 * rails db:seed
